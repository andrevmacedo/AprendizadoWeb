# 📘 Atividades Iniciais em HTML e CSS

Este repositório contém um conjunto de atividades desenvolvidas com o objetivo de **aprender e praticar os conceitos básicos de HTML e CSS**.

As atividades foram propostas na disciplina **Scripts de Programação**, ministrada pelo professor **Paulo Fratta**, e abordam diferentes formas de estilização e estruturação de páginas web.

---

## 🎯 Objetivo

O projeto tem como finalidade:

* Compreender a estrutura básica do HTML
* Aplicar diferentes formas de estilização com CSS:

  * CSS **inline**
  * CSS **interno**
  * CSS **externo**
* Desenvolver páginas simples para fixação dos conceitos iniciais

---

## 📂 Estrutura do Projeto

O repositório está organizado em diferentes atividades:

### 📁 Portfólio - HTML

* Criação de uma página de portfólio pessoal
* Uso de imagens e organização de conteúdo
* Aplicação de estilos básicos

---

### 📁 Cardápio - HTML

* Desenvolvimento de um cardápio digital
* Inserção de imagens e descrição de produtos
* Estruturação de layout simples

---

### 📁 Cartão LinkedIn - HTML

* Criação de um cartão de perfil profissional
* Uso de informações pessoais
* Estilização visual inspirada em redes profissionais

---

### 📁 Portal de Notícias - HTML

* Simulação de um portal de notícias
* Uso de **CSS externo** (`estilo.css`)
* Aplicação de layout com múltiplas seções
* Estilização de textos, títulos e blocos de conteúdo

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3

---

## 📚 Conceitos Praticados

* Estrutura básica de páginas HTML
* Uso de tags semânticas
* Inserção de imagens
* Organização de conteúdo
* Estilização com:

  * Inline (`style=""`)
  * Interno (`<style>`)
  * Externo (`.css`)
* Cores, bordas e espaçamentos

Este projeto faz parte do processo de aprendizado e evolução no desenvolvimento web, servindo como base para projetos mais avançados no futuro.
